TicTacToe Pseudo Code

1. Draw a board
2. Players interact with board on click
3. When board is clicked, the empty box is replaced with an 'X' or an 'O'


4. Premise: 2 Player Game
	- First player plays, evaluate board results
	- Second player plays, evaluate board results etc.
	- Continue playing until a winning combo is found
		- if a winning combo is not found, check if there is a tie (aka if the board is all filled up with X's and O's)



5. Extras:
- display on screen who's turn it is.
- ask for user's name, refer to player by user name
- when someone wins, announce who won
